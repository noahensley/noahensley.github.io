02 project

Compiler-Sharp.

compiler-sharp

A hand-written C# compiler using the shunting-yard algorithm for expression parsing. Input source code passes through a regex-driven lexer, a recursive-descent parser that builds a full typed AST, and a type checker with comprehensive test coverage. Output modes include JSON token streams, JSON AST, Graphviz dotfiles, and box-drawn tree diagrams.
